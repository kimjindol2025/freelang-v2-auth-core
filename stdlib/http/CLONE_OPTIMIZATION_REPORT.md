## 🛠️ 구현된 최적화

### 1️⃣ 메모리 풀 (Object Pooling)
- 객체를 재사용하여 GC 오버헤드 제거
- 메모리 할당 오버헤드 제거
- 메모리 조각화 방지

### 2️⃣ 필드 압축 (Field Compression)
- 106 bytes → 14 bytes (7.6배 감소)
- App 매핑: 0-3 인덱스 (문자열 제거)
- 상태: 0/1 (문자열 제거)
- 불필요한 필드 제거
